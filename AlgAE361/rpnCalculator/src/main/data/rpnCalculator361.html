RPN calculator

Quick instructions:

  • Start the calculator from the Algorithm menu.
    • When prompted, give an expression to evaluate.
    • Must be in RPN notation, e.g., 2 3 * 1 +
      • positive integers
      • operators (+ - * /) work on two most recent values
      • 'p' to print result
      • 'q' to quit the calculator
  • Use the speed control to slow down the animation if desired.
  • Alternatively, use the pause button.
    • When execution pauses inside of an algorithm, press the "step" button to single-step through the code or use the "play" button to resume continual execution.
    • If you find the display is too hopelessly tangled, you can use your mouse to drag the nodes to more appropriate positions whenever you are paused within a running algorithm.
  • If the program does not run (you may need to click on the box below to authorize it), download the Jar file directly and run it on your local PC.

More detailed help is available from the Help menu.

This demo was prepared with the AlgAE animation engine.
